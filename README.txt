ABS v2 preview

Это очищенная демо-версия по согласованной логике:
- Проекты
- Карточка проекта
- Бриф без автоанализа
- Статистика Авито с выбором метрик и CSV/Excel-выгрузкой
- Сайт: заглушка аналитики и админ-панели
- Калькулятор
- Настройки
- База знаний

Данные хранятся в localStorage только для просмотра прототипа.
В боевой версии нужно подключить backend, БД, авторизацию, защищённое хранение API-ключей и настоящую синхронизацию Авито/сайта.
